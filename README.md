## Todo List

### Используемый стек:

- React;
- TypeScript;
- React Hook Form;
- Redux Toolkit;
- IndexedDB;
- SCSS;
- Material UI.

### Установка:

```js
npm install
```

### Команда для запуска в dev-режиме:

```js
npm run start
```

### Команда для build-сборки:

```js
npm run build
```
